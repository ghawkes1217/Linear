Module 12: Null Space

The null space of a matrix \(A\) is the set of all vectors \(x\) such that \(Ax=\mathbf{0}\) where by \(\mathbf{0}\) we mean the vector of all \(0\)s (of the same height as the matrix \(A\)). In other words, the coordinates of \(x\) specify a linear combination of the columns of \(A\) that is equal to \(0\).

One way to find vectors in the null space, or equivalently, a linear combination of the columns of \(A\) that gives \(\mathbf{0}\), is as follows. First, apply row operations until the augmented matrix associated to \(Ax=\mathbf{0}\) is in reduced row echelon form. Then, identify which columns have pivots (we'll call these pivot columns) and which columns do not have pivots (we'll call these free columns). Then choose a particular free column. Take \(1\) times that column and \(0\) times all other free columns. At this point you can deduce what the rest of the coefficients (in other words the coefficients for the pivot columns) need to be in order for the resulting linear combination to be \(\mathbf{0}\).

The vectors attained in this way we will call special vectors and they generate the null space in the sense that one can find any element of the null space by taking a certain linear combination of them. Since we constructed one special vector for each free column this means that the total number of special vectors is equal to the number of free variables (this number is known as the dimension of the null space).

In this module, you are required to apply row operations to get the shown augmented matrix into reduced row echelon form. Once this is done, you will need create the special vectors. In the lower levels you will have considerable help with this and only need to fill in a few values. However, (on average) as the difficulty increases, the burden of finding the entries of the special vectors becomes more and more your responsibility. If no input boxes appear under the "Special Vectors" heading, then there are no special vectors and so all you need to do is find the reduced row echelon form.
